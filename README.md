# 💧 Drink Water Annoying App

A simple but aggressive desktop reminder application designed to **force you to stay hydrated** — even when you’re too focused to notice notifications.

---

## 🚀 The Idea

Most reminder apps fail because users get used to notifications and start ignoring them.

This project was built to solve that problem by introducing **forced interaction**.

Instead of just reminding you to drink water, this app **escalates actions** until you respond.

---

## ⚙️ How It Works

- ⏱️ Sends a reminder every 30 minutes  
- 🔔 If ignored → repeats warnings continuously  
- ⚠️ If still ignored → **forces your PC to go to sleep**  

The goal is simple:  
> Break your focus just enough to make you take action.

---

## 🛠️ Tech Stack

- **Language:** C#  
- **Framework:** Windows Forms (.NET)  

---

## 🔍 Key Features

- Timed water reminders  
- Escalating alert system  
- System-level action (PC sleep)  
- Lightweight and simple design
